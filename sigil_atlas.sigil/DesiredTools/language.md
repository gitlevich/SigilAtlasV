# Desired Tools

A temporary place to keep track of tools that would help shape and maintain this spec.

Right now I want:

- a tool to compile or validate the sigil tree
- a tool to check broken, dangling, or shadowed references
- a tool to show diffs after writes, moves, renames, and deletes
- a tool to propose and possibly execute batch tree reshapes safely
- a tool to confirm that a write really changed the file system
- a tool to inspect unresolved conceptual mismatches between language and structure
