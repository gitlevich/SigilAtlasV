# Name

What gets resolved. A name is a token that refers to a @Sigil, an @EcologicalPsychology@Affordance, or an @Invariant within a @Scope.

A name is bound when it is defined in a @Scope — it acquires meaning there. A name #shadows an outer binding when redefined in an inner @Scope: the inner meaning prevails locally.

A name is unresolved when no @Scope in the resolution chain contains a binding for it — a gap in the domain language.
