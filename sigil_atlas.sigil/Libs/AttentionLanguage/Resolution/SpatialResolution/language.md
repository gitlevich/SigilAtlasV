# Spatial Resolution

How finely @attention distinguishes structure within a @frame. The number of @contrasts discriminated simultaneously.

In a @sigil tree, spatial resolution = leaf granularity. Each decomposition level adds a degree of freedom — more joints on the finger, more precision in the grasp.

Spatial resolution caps articulation. Walking a thrown taxonomy's root-to-leaf paths past this ceiling yields distinctions the @POV cannot discriminate from each other at the current viewport, so further articulation is wasted. The articulator should walk each dropped @sigil exactly to the current ceiling and no deeper. @Navigation zoom raises the ceiling locally in the viewed region, which permits deeper articulation there without touching the rest of the surface.
