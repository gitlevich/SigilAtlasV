# Sigil Atlas

I use this application to @explore a @corpus of @images.

I can:

- define a subset of the @corpus I want to work with - a @slice
- see the @slice ordered by @time
- see the @slice arranged into @neighborhoods by @similarity
- look for @images similar to a @targetImage
- save any beautiful or interesting collage I see on the screen along with their @slice definition
- load a saved collage
