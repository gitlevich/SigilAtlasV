# Image File

An image of this @Corpus #identified-by its @identity derived from bytes. 

Lives on a filesystem. 

The image file !accepts-jpeg, !accepts-png, !accepts-heic formats.

Each @image !has-a-high-rez-preview so it could be rendered in resolution high enough to be crisp when zoomed into. It is stored in @Workspace@ImageCache.
