# TimeScale

The gap threshold that determines how @story cluster. Controls the granularity of temporal grouping:

- Minutes: each burst of rapid shooting is a story.
- Hours: each session (morning walk, afternoon shoot) is a story.
- Days: each day of shooting is a story.
- Weeks/months: each trip or project is a story.

Adjustable in real time. Changing @timescale recomputes @story and reflows the @strip layout.
