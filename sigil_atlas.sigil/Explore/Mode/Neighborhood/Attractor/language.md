# Attractor

The point on the @torus around which images arrange. Set by pointing at an @image or by accumulated #like and #dislike signals.

When I #like an image, the attractor moves toward it. When I #dislike, it moves away. Multiple signals compose — the attractor is a running summary of my expressed taste.

When I #find-similar, #find-opposite, #find-nearby, or #find-same-time on an image, that image becomes the attractor and the @distanceMetric switches accordingly.
