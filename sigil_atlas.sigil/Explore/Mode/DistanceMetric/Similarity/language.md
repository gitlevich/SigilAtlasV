# Similarity

The inverse of @distanceMetric. Two @images are similar when their distance is small. What counts as similar depends on which metric is active — embedding proximity, temporal proximity, physical proximity, or a composite.

@SpaceLike arranges the @torus so that similar images are neighbors everywhere. @TasteLike arranges it so that images similar to the @attractor are closest to it.
