# Status Bar

Bottom edge of the @screen. Shows the current @mode (timelike or spacelike), @slice size — how many @images match the current @RelevanceFilter — @PointOfView coordinates and zoom level, and the active @RelevanceFilter description if any.

At @ingest time, shows real-time stats of ingest, with options to pause the run or resume a paused run.
