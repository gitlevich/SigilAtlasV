# Thing

Anything that I can name. Flower. Person. Green jacket. Bug.

It is, of course, a @sigil: #named and !recognizable.

A @thing can be primitive — a leaf name — or compound. A compound @thing is a decision tree: each constituent is a discrimination step refining which leaf applies. Beetles, butterflies, flies, dragonflies → insects; insects, spiders, ... → bugs. Each #compose goes one level up in abstraction.

An @image satisfies a compound @thing if it satisfies any leaf reachable by descent. The tree is embedded structurally, not collapsed to a flat name, because its structure is what identifies the @thing to the last degree of freedom needed to distinguish it.
