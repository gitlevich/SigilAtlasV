# Thing

Anything that I can name. Flower. Person. Green jacket. Bug.

It is, of course, a @sigil: #named and !recognizable.

A @thing can be primitive — a leaf name — or compound, holding constituent @things any of which satisfies it. A compound aggregates neighbor @things at the same scale: beetles, butterflies, flies, dragonflies → insects; insects, spiders, ... → bugs. Each #compose goes one level up in abstraction.
