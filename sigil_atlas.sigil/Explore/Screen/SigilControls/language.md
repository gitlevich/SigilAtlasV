# Sigil Controls

The @rightPanel. A @sigil builder — define the region of @contrastspace you want to inhabit.

A sigil is a set of @contrastControls with @bandPass ranges. The active set IS the query. The @slice is the result.

Also contains: @mode toggle, @timescale slider, @stripHeight slider.
