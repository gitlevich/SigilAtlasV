# Status Bar

Bottom edge of the @screen. Shows:
- Current @mode (timelike / spacelike / tastelike)
- @slice size: how many @images match the current @RelevanceFilter
- @PointOfView coordinates and zoom level
- Active @RelevanceFilter description (if any)


At @ingest time, shows real-time stats of ingest, with options to pause the run / resume paused run.
